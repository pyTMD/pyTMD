==============
nearest_extrap
==============

- Uses `kd-trees <https://docs.scipy.org/doc/scipy/reference/generated/scipy.spatial.cKDTree.html>`_ for nearest-neighbor (`NN`) extrapolation of valid tide model data

----------------

.. code-block:: python

    extrap = nearest_extrap(x,y,data,XI,YI)
